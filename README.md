# 🚀 AdubamOS

**AdubamOS** is a lightweight, experimental terminal-based operating environment written in Python.  
It simulates a minimal OS experience with its own shell, package manager, and modular system.

---

## 🔗 Repository

👉 https://github.com/gagafik-sus/test

---

## ✨ Features

### 🖥️ Custom Shell
- Interactive shell (`Adubam Shell`)
- Built-in commands system
- Simple and extendable architecture

### 📦 Package Manager
- `apm` (Adubam Package Manager)
- Install and manage custom apps
- Lightweight and fast

### 🧩 Modular System
- Supports apps via `/apps` directory
- Optional mods support (`/mods` if present)
- Expand functionality without touching core logic

### ⚙️ Built-in Tools
- Calculator
- Clock
- File management commands
- System info (`fetch`)

---

## 📁 Project Structure

```
AdubamOS/
├── apps/        # Applications
├── mods/        # Optional mods (may not exist)
├── main.py      # Entry point / core logic
```

---

## 🚀 Getting Started

### Requirements
- Python 3.x

### Run

```bash
python main.py
```

## 💡 Commands

```
help    - show help
calc    - calculator
clear   - clear screen
system  - system commands
fetch   - system info
clock   - show time
mkdir   - create directory
delete  - delete files
file    - file operations
apm     - package manager
q       - exit
```
